📱 Prototipo App Mercatino - UX/UI Optimization
Questo progetto nasce dalla volontà di modernizzare l'esperienza utente (UX) e l'interfaccia grafica (UI) dell'app ufficiale del franchising Mercatino. L'obiettivo è offrire uno strumento più fluido, professionale e immediato per i clienti.

🚀 Caratteristiche Principali
Navigazione Avanzata: Struttura nativa in Java con Navigation Drawer e Bottom Navigation Bar per un passaggio rapido tra Home, Negozi e Card.

Digital Card System: Generazione di codici a barre ottimizzata per la scansione istantanea tramite i lettori laser presenti nei punti vendita.

User Profile: Gestione profilo personalizzata con caricamento foto da galleria e salvataggio persistente (SharedPreferences).

Search Experience: Home screen con barra di ricerca e filtri a icone per una consultazione rapida dell'inventario.

🛠 Tech Stack
Linguaggio: Java (Android Nativo)

IDE: Android Studio

Componenti: Material Design, Fragments, Permission System (GPS/Storage), SharedPreferences.
